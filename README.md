# MyMaxxio
Android-based Community Information System for GranMax Luxio Club Indonesia (Admin App).

## Features
- Member Management
- Event Management
- Event Post Management
- Announcement Management
- Members Cash Management (Uang Kas)
- Sub-community (Chapters) Cash Management (Uang Kas)

## Technologies Used
- Android Studio (Java)
- Firebase (Firestore, Storage, Authentication)
